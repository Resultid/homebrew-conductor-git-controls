Git controls for
Conductor.

A native macOS Git client built for Conductor. Stage, commit, and push across all your workspaces from one interface.

$ brew install --cask Resultid/conductor-git-controls/conductor-git-controls

Why Git Controls

Purpose-built for Conductor workspaces.

Conductor Native

Reads directly from Conductor's workspace database. Every workspace you create in Conductor appears automatically — no setup required.

High-Speed Staging

Bulk stage, commit, and push in one click. Switch between Conductor workspaces and manage changes without leaving the app.

Native Performance

Built with Tauri and Svelte 5 for a footprint smaller than 10MB. Instant startup, zero Electron bloat.

Interface

All your Conductor workspaces. One sidebar.
